nhk-gogaku-streaming
====================
NHK語学番組のストリーミングを再生できるアプリ。

音声ファイルの保存は現在実装中です。（なかなか難しいですね。暗号化されているMpeg2 TSファイルを復号化して、1つのmp4ファイルに変換しましたが、生成したファイルをmacからffmpegでmp3に変換するところでエラーになります）
